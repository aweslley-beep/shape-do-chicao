SHAPE DO CHICÃO - versão aberta sem senha

Como usar:
1. Abra index.html no navegador ou hospede a pasta na Vercel.
2. No iPhone, abra o link no Safari e toque em Compartilhar > Adicionar à Tela de Início.
3. Cada pessoa informa nome, idade, peso, altura, objetivo e nível.
4. O app calcula IMC, peso ideal, água, calorias, proteína, sono e gera treino/dieta/dicas.

Importante:
- Não possui senha nem login.
- Os dados ficam salvos apenas no aparelho/navegador de cada usuário.
- Para banco de dados em nuvem por usuário, integre Firebase ou Supabase.
